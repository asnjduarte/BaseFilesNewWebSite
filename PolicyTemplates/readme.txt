The PolicyBaseFileSql.php should be run first. You will need to manually change the database settings. Place the file on your root project folder.
1. Type the following in a new visual studio code terminal: php PolicyBaseFileSql.php

The AddGenericPolicyBaseFile.php should be run second. 
1. Type the following in a new visual studio code terminal: php AddGenericPolicyBaseFile.php
2. You will need to manually add the two pieces of code to the php and class area of the \model\commonFunctions\Factory.php after the file is run.

Afterwards you may run the other policies that you may be interested in.
